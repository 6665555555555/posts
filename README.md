# مشروع تجريبي: واجهة تسجيل دخول وإنشاء حساب (عميل فقط)

محتوى المشروع:

- `index.html` — صفحة الواجهة الرئيسية تحتوي على نماذج التسجيل وتسجيل الدخول.
- `css/style.css` — أنماط بسيطة للصفحة.
- `js/app.js` — منطق التحقق من صحة النموذج، تشفير كلمة المرور (SHA-256)، وحفظ المستخدمين في `localStorage`.

ملاحظات مهمة:

- هذا تطبيق تجريبي يعمل على جهة العميل فقط (بدون خادم). المستخدمون والحسابات مخزنة محلياً في متصفحك باستخدام `localStorage`.
- لأغراض الإنتاج يجب استخدام خادم آمن، قاعدة بيانات، وتخزين كلمات المرور بتجزئة آمنة مع ملح (مثلاً باستخدام bcrypt) وعدم الاعتماد على `localStorage`.

كيفية التشغيل:

1. افتح الملف `index.html` في متصفح حديث (Chrome/Edge/Firefox).
2. استخدم واجهة "إنشاء حساب" لعمل حساب جديد.
3. بعد الإنشاء، سجّل الدخول بنفس اسم المستخدم أو البريد/الهاتف وكلمة المرور.

التحقق من متطلبات كلمة المرور (مطابقة لمتطلباتك):

- حرف كبير واحد على الأقل
- حرف صغير واحد على الأقل
- رقم واحد على الأقل
- رمز واحد على الأقل
- الطول 8 أحرف على الأقل
